A partir de una frase, cambiar todas las apariciones de un conjunto de palabras por otro conjunto de palabras.

ejemplo:

frase: El real madrid es el mejor.

buscar: real madrid
cambiar: sporting de gijon


salida:
El sporting de gijon es el mejor.
